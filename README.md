# Codveda Java Tasks

## Project Description
This repository contains Java projects developed as part of Codveda internship tasks.

## Projects Included

### Basic Calculator
- Performs addition, subtraction, multiplication, and division operations.

### Employee Management System
- Add and display employee details.

### Banking Application
- Perform deposit, withdrawal, and balance checking.

### Number Guessing Game
- Guess a randomly generated number.

### Factorial Calculation
- Calculates factorial using recursion.

### File Handling Demo
- Demonstrates reading and writing files in Java.

## Technologies Used
- Java
- OOP Concepts
- Eclipse IDE
