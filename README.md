# Multithreaded Chat Application

## Description
This is a console-based chat application developed using Java Sockets and Multithreading. The application allows multiple clients to connect to a server and exchange messages in real time.

## Features
- Multiple clients can connect simultaneously.
- Uses multithreading to handle multiple users.
- Broadcasts messages to all connected clients.
- Console-based application using Java.

## Technologies Used
- Java
- Java Sockets
- Multithreading
- Object-Oriented Programming (OOP)

## Project Structure
```
ChatApplication.java
```

## How to Run

### Compile the Program
```bash
javac ChatApplication.java
```

### Start the Server
```bash
java 'ChatApplication$Server'
```

### Start the Client
Open another terminal and run:

```bash
java 'ChatApplication$Client'
```

You can open multiple terminals and run multiple clients to test the chat application.

## Concepts Used
- Socket Programming
- ServerSocket and Socket Classes
- Threads
- Input/Output Streams
- Exception Handling

## Output
- Server accepts multiple client connections.
- Messages sent by one client are displayed to all connected clients.
